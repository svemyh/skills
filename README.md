# svemyh/skills

Installable agent skills for Cursor, Codex, Claude Code, and other SKILL.md-compatible agents.

## Install

```bash
# Install and choose skills interactively
npx skills add svemyh/skills

# Install this specific skill
npx skills add svemyh/skills --skill agentic-oriented-programming
```

## Included skills

- `agentic-oriented-programming` - coding doctrine for agent-era software design: locally simple, globally constrained.
